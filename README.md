# HEOS

HEOS 一个为 HE 服务器开发的模组，或者说我个人是把它当做系统来做的，因此它有着各种各样的功能，功能内容如下（部分）。

---

## 认证相关

### 登录系统

我知道这并没有什么亮眼的地方，但是我们的登录系统是可以分离离线和正版玩家的，这可以解决相当多的麻烦。

---

## 服务器状态显示

### Tab TPS

它可以让你在玩家列表里面看到当前服务器的 TPS 和 MSPT，有助于了解当前服务器的运行情况。

---

## 管理功能

### 自定义封禁

该功能允许你封禁玩家的时候加上时长，到时间后会自动解封。

### 玩家数据迁移

你可以把一个玩家的数据迁移给别的玩家，如果你在使用本模组前已经有了一些玩家的数据，你可以用此功能进行迁移数据。

---

## 日志优化

### 更好的日志

如果你是个生电服务器，那么该功能可以帮你过滤掉制作更新抑制时后台一直在刷屏的日志，虽然说实际上只是把很多信息缩短成了一行，但是他确实好的多。
