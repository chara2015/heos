# Heos 快速开始指南

## 系统要求

- Java 17+
- Minecraft 1.20+
- Fabric Loader 0.15.0+
- Fabric API

## 构建项目

```bash
# 清理缓存
./gradlew clean

# 构建
./gradlew build
```

构建完成后，JAR文件位于：`versions/1.21.2/build/libs/`

## 服务器配置

### 1. 启用正版验证

编辑 `server.properties`：
```properties
online-mode=true
```

### 2. 安装mod

将构建的JAR文件放入服务器的 `mods` 文件夹

### 3. 启动服务器

```bash
java -jar server.jar
```

## 玩家使用指南

### 正版玩家

1. 使用正版Minecraft客户端
2. 直接连接服务器
3. 自动通过验证，无需额外操作
4. 看到消息：`§a欢迎回来，正版玩家！`

### 离线玩家

#### 用户名要求
离线玩家的用户名必须包含特殊字符，例如：
- `Player+123`
- `Test.User`
- `User-Name`
- `My_Player.2024`

#### 首次连接（注册）

1. 连接服务器
2. 看到提示：
   ```
   =================================
   请使用 /register <密码> <确认密码> 注册
   =================================
   ```
3. 执行命令：
   ```
   /register mypassword mypassword
   ```
4. 注册成功后自动登录

#### 后续连接（登录）

1. 连接服务器
2. 看到提示：
   ```
   =================================
   请使用 /login <密码> 登录
   =================================
   ```
3. 执行命令：
   ```
   /login mypassword
   ```
4. 登录成功

## 命令列表

| 命令 | 说明 | 权限 |
|------|------|------|
| `/register <密码> <确认密码>` | 注册账号 | 所有离线玩家 |
| `/login <密码>` | 登录账号 | 所有离线玩家 |

## 常见问题

### Q: 为什么我的离线玩家无法连接？
A: 确保用户名包含特殊字符（如`+`、`-`、`.`），纯字母数字用户名会被识别为正版玩家。

### Q: 正版玩家需要注册吗？
A: 不需要，正版玩家通过Mojang官方验证后自动登录。

### Q: 忘记密码怎么办？
A: 当前版本需要管理员手动重置（删除玩家数据）。

### Q: 密码安全吗？
A: 当前版本密码以明文存储在内存中，仅用于测试。生产环境需要实现密码哈希加密。

### Q: 数据会保存吗？
A: 当前版本数据存储在内存中，服务器重启后会丢失。需要实现数据库持久化。
