# Heos 认证系统命令

## 玩家命令

### 注册账号
```
/register <密码> <确认密码>
```
- 首次加入服务器时使用
- 密码长度：4-32 个字符
- 密码使用 PBKDF2 哈希加密存储

### 登录
```
/login <密码>
```
- 每次加入服务器时使用（如果未自动登录）

### 修改密码
```
/changepassword <旧密码> <新密码> <确认新密码>
```
或
```
/changepw <旧密码> <新密码> <确认新密码>
```
- 需要输入旧密码验证身份
- 新密码长度：4-32 个字符

## 管理员命令

需要 OP 等级 3 或以上

### 重置玩家密码
```
/heosadmin resetpassword <玩家名> <新密码>
```
- 用于玩家忘记密码时
- 如果玩家在线，会收到新密码通知

### 注销玩家账号
```
/heosadmin unregister <玩家名>
```
- 删除玩家的注册信息
- 如果玩家在线，会被踢出服务器

### 查看玩家信息
```
/heosadmin info <玩家名>
```
- 显示玩家的注册信息
- 包括 UUID、IP、注册时间、最后登录时间等

## 数据存储

- 玩家数据保存在 `<服务器目录>/heos_data/` 文件夹
- 每个玩家一个 JSON 文件：`<玩家名小写>.json`
- 密码使用 PBKDF2-HMAC-SHA256 哈希算法加密
- 格式：`salt:hash`（Base64 编码）

## 安全特性

1. **密码哈希**：使用 PBKDF2 算法，10000 次迭代
2. **随机盐值**：每个密码使用独立的随机盐值
3. **防时序攻击**：密码比对使用常量时间算法
4. **自动保存**：所有数据变更自动保存到磁盘

## 玩家类型

### 正版玩家
- 自动通过 Mojang 验证
- 无需注册和登录
- 用户名必须符合 Mojang 规范（字母、数字、下划线）

### 离线玩家
- 用户名包含特殊字符（如 `-`、`+`、`.`）
- 需要注册和登录
- 密码保存在本地服务器
